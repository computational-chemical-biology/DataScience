Exercício-001#

Importe o conjunto de dados diretamento do github em um DataFrame - https://raw.githubusercontent.com/CongLiu-CN/zero-to-mastery-ml/master/data/heart-disease.csv

Exercício-002#

Crie duas variáveis, uma para armazenar a resposta de predição (coluna target) e outra com todas as outras variáveis, exceto target, que serão utilizadas como variáveis explicativas.

Exercício-003#

Use o método train_test_split dentro do módulo model_selection da biblioteca sklearn e divida os dados em conjuntos de treinamento e teste.

Exercício-004#

Importe o classificador RandomForestClassifier da biblioteca sklearn e criem uma variável com uma instância do classificador. Dica: veja o manual do classificador.

Exercício-004#

Ajuste um modelo de classificação com os dados de treinamento.

Exercício-005#

Execute a predição da variável reposta com o conjunto de teste de variáveis explicativas.

Exercício-006#

Avalie as predições do modelo no conjunto de treinamento e no conjunto de testes.

Exercício-007#

Repita as etapas executas acima para os modelos de classificação listados abaixo e compare os resultados.

  • LinearSVC

  • KNeighborsClassifier

  • SVM

  • LogisticRegression

  • RandomForestClassifier

Exercício-008#

Crie um DataFrame com os resultados do exercício anterior e faça um gráfico de barras a partir do DataFrame.

Exercício-009#

Utilize o modelo LogisticRegression para selecionar parâmetros do modelo que podem ser alterados para testar o desempenho do modelo. Exemplo.

Pode se usar GridSearchCV ou RandomizedSearchCV. Utilize o conjunto de treinamento das etapas anteriores.

Exercício-010#

Mostre os melhores parâmetros encontrados na comparação, e o calcule a acurácia da predição com o modelo para estes parâmetros no conjunto de teste.

Exercício-011#

Calcule a matriz de confusão para o modelo e obtenha as seguinte métricas (calculadas com a matriz de confusão e com métodos de sklearn): Precisão, especificidade, sensibilidade, score f1 e Área Abaixo da Curva ROC, incluindo a curva;

Exercício-012#

Execute e interprete a validação cruzada com o método cross_val_score do módulo model_selection.

Exercício-013#

Exporte e importe um modelo treinado utilizando a biblioteca joblib. Obtenha a acurácia do modelo treinado nos dados de teste.
