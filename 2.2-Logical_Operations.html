Operadores Relacionais#

Símbolo

Operação

==

True, se operandos são iguais

!=

True, se operandos não são iguais

<

menor que

>

maior que

<=

menor ou igual a

>=

maior ou igual a

 
a = 21
b = 10
c = 0

a == b

False

a != b

True

a < b

False

a <= b

False

b >= a

False

Operadores Lógicos#

Símbolo

Operação

and

Se ambos operandos são verdadeiros a condição se torna verdadeira

or

Se algum dos operandos é verdadeiro a condição se torna verdadeira

not

Usado para reverter o estado lógico do operando

 
a > b and a > c

True

a == 0 or c == 0

True

a == 0 or b == 0

False

not a > b and a > c

False

Operadores de Conjunto#

Símbolo

Operação

in

Retorna verdadeira (True) se o operador do lado esquerdo estiver contido no operador do lado direito

not in

Retorna falso (False) se o operador do lado esquerdo estiver contido no operador do lado direito

 
'P' in 'Python'

True

'Y' in 'Python'

False
